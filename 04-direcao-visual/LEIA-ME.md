# 04-direcao-visual — O que é e como usar

`direcao-cliente.md` é a direção de design system escrita pelo Diretor de
Arte do LUVRE: personalidade visual, sistema de cor derivado das cores da
marca, par tipográfico (com links de download e licença), direção de
componentes, elemento-assinatura e motion.

**Não é CSS pronto.** É a direção pro Claude Design CONSTRUIR o design
system — ele tem liberdade de execução dentro dela. Se alguma escolha da
direção conflitar com uma referência de execução de `03-referencias/`,
a direção manda na ESTÉTICA e a referência manda no NÍVEL DE EXECUÇÃO.
